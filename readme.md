## Environment
Set `YEAR` for year and `MACHINE` for your machine (used in naming benchmark-files) in `.env`

## Stub-Generator

    bin/aoc generate <day> <year=current year>
    

## Run solution

    bin/aoc run <day> [<to_day>] 
    bin/aoc run <day> <year>
    bin/aoc run <day> <to_day> <year>

## Run Benchmarks

    bin/aoc bench
